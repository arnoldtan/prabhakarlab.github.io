RCA - Reference Component Analysis

Reference Component Analysis (RCA) is a semi-supervised clustering algorithm that annotates cells in single-cell RNA-sequencing data by correlation to bulk reference datasets of pure cell types.

DFilter

DFilter is a generalized signal detection tool for analyzing next-gen massively-parallel sequencing data by using ROC-AUC maximizing linear filter.

Hence it is an ideal tool for detecting peaks in tag-profile of ChIP-seq, DNase-seq, FAIRE-seq, ATAC-seq, MNase-seq, RIP-seq, CLIP-seq, ChIP-exo, Sono-seq etc.
